# 🔐 encrypt_pan — 私密加密网盘

> 零元数据 · 异或加密 ·  隐私优先

---

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| **🔒 XO 异或加密** | 文件内容经 XOR 加密存储，服务端不保存原始数据 |
| **🕳️ 零元数据存储** | 不记录文件名、类型、大小等敏感信息，仅保留加密后的二进制 |
| **📛 固定临时文件名** | 所有文件以固定规则命名，无法从文件名推断内容 |
| **⬇️ 强制下载** | 禁止浏览器直接预览，必须下载到本地解密 |
| **🔑 上传口令验证** | 无 Session/Cookie，一次性 Token 内存存储，刷新即失效 |
| **📋 保密文件列表** | 文件列表同样加密，服务端无法获知存储内容 |
| **🔐 双模式登录** | 宽容模式（口令）/ 严格模式（邮箱验证码） |
| **📧 PHPMailer 集成** | 支持 SMTP 邮件验证码登录 |

---

## 🚀 快速开始

### 环境要求
- PHP 7.4+
- MySQL / MariaDB
- SMTP 邮箱（严格模式需要）

### 安装

1. 上传代码到服务器
2. 访问 `/install/` 进入安装向导
3. 配置数据库、选择模式、设置管理员口令
4. 安装完成后自动锁定，展示 lock 内容

```bash
# 目录结构
encrypt_pan/
├── index.php          # 主页（上传/下载）
├── admin/
│   └── index.php      # 管理后台
├── install/
│   └── index.php      # 安装向导
└── ...
```

## 🔧 使用说明

### 上传文件
1. 输入上传口令 → 获取临时 Token
2. 选择文件 → 自动 XOR 加密上传
3. 获得下载链接

### 下载文件
1. 访问下载链接
2. 输入解密密钥（或管理员口令）
3. 文件解密后强制下载到本地

### 管理后台
- 宽容模式：admin 口令登录
- 严格模式：邮箱验证码登录（60秒冷却）
- 功能：上传口令开关、更改 admin 口令、查看日志

## 🛡️ 安全设计

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   原始文件   │ → XOR密钥加密 → │  密文存储    │ → 强制下载输出 │
│  (客户端)   │     │  (服务端)   │     │  (浏览器)   │
└─────────────┘     └─────────────┘     └─────────────┘
         ↑_________________________________↓
              密钥仅由用户持有，服务端不保存
```

- **服务端零知情**：不知道存了什么、文件名是什么、谁上传的
- **一次性 Token**：上传权限验证后即刻失效，无持久化凭证
- **无预览漏洞**：所有文件强制 `Content-Disposition: attachment`，防止浏览器直接打开
